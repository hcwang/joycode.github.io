我是一只小小鸟

Posted on 二 14 十月 2003 in 业界评论

[ “Joyeasy=OVER"]{lang="EN-US"}这是我刚才收到[summer]{lang="EN-US"}的十二个字，我不知道说些什么好。[]{lang="EN-US"}

[ ]{lang="EN-US"}这几天我一直为网站的宕机率而难过，好几天没有上来过，闹得[“]{lang="EN-US"}好开心[”]{lang="EN-US"}姐姐还发贴子找我。[]{lang="EN-US"}

[ ]{lang="EN-US"}事情很简单，[summer]{lang="EN-US"}想在自己的北京所有站上用一下这个网站的论坛，我考虑过之后，给他在冒名的地址上传了一个加密版本（敝帚自珍吧），结果冒名不支持。于是我让他用现成的流言，休尚，大师，天下这几个版面先支撑着，等换了[sql]{lang="EN-US"}之后再给他开新版，可是[summer]{lang="EN-US"}说这几个版主[“]{lang="EN-US"}够戗[”]{lang="EN-US"}，不愿意接受。[]{lang="EN-US"}

[ ]{lang="EN-US"}结果[summer]{lang="EN-US"}下午的时候用[OICQ]{lang="EN-US"}告诉我在[“]{lang="EN-US"}流言[”]{lang="EN-US"}上发了一个贴子，让我来看看，如果不合时宜就删掉它。我看了之后，十分伤心。[]{lang="EN-US"}

[ “]{lang="EN-US"}伤心[”]{lang="EN-US"}，这就是我现在的感觉，我不知道回些什么好，试着写了十多次，可是总感觉好象是一群小孩在那儿打架，争论是谁的错似的。[]{lang="EN-US"}

下班后与同事专门去喝了些酒，凑着酒劲又写了几次，可是写到一半还是伤心，有一种绞心似的疼痛。我真的不想写，我也曾经想过干脆就给他删掉就是了，我为什么那么[“]{lang="EN-US"}故作高尚[”]{lang="EN-US"}得给他加至精华区？[]{lang="EN-US"}

[ ]{lang="EN-US"}突然想到我在上周末的时候冒出的一个想法：我是一只小小鸟。这周一直想用这个名字写个贴子，却不知道写些什么，现在就在这儿发些牢骚吧。[]{lang="EN-US"}

[ ]{lang="EN-US"}毕业前，按照[summer]{lang="EN-US"}说的，我想成立一个[ASP]{lang="EN-US"}方面的学习网站，之所以叫[“]{lang="EN-US"}互助联盟[”]{lang="EN-US"}，就是因为开心也是刚学，仅知道一些皮毛，希望大家互相帮助。[Summer]{lang="EN-US"}那天恰好回校，于是一起聊了起来，越说越投机，于是我们当天就起草了章程之类的东西（我在校期间，创建过校计算机协会，我任会长，[summer]{lang="EN-US"}是外联部长，现在同学打电话给我，还叫我[“]{lang="EN-US"}会长[”]{lang="EN-US"}，当时办联盟的时候，也按着这条路走了下来）。[]{lang="EN-US"}

[ ]{lang="EN-US"}大家如果去过当时个人网站时期的联盟的话，就知道当时我们的窘境了，就我编的一个不入流的论坛（当时连[HTML]{lang="EN-US"}都没有屏蔽）及[summer]{lang="EN-US"}的一个友情链接系统，除此之外，就是章程之类的东西。[]{lang="EN-US"}

[ ]{lang="EN-US"}后来[C.T.]{lang="EN-US"}及[Blahblah]{lang="EN-US"}，[ren,]{lang="EN-US"}踏雪无痕，[Lucy]{lang="EN-US"}等加入了进来。网站慢慢得强大了起来，当然了，现在还很弱小，距离我的梦想还很远，有了聊天室及其它一些栏目，于是我不顾[summer]{lang="EN-US"}的劝阻，租了虚拟主机，准备大干一场，夜里经常到三点多，白天在公司也心不在蔫，凑空就上来溜一下。可是随着时间的推移，[blahblah]{lang="EN-US"}由于远在加拿大，而且我们的[IP]{lang="EN-US"}加拿大地区不好访问，[ren]{lang="EN-US"}则调动了工作，失去了上网的机会，踏雪忙于电脑报的栏目制作，[lucy]{lang="EN-US"}忙着本公司的电子商务，大家都逐渐失去了联系。原来号称十大核心成员的联盟就剩下我与[c.t.]{lang="EN-US"}及[summer.]{lang="EN-US"}。[]{lang="EN-US"}

[ ]{lang="EN-US"}很突然的，[Summer]{lang="EN-US"}说要做[“]{lang="EN-US"}所有[”]{lang="EN-US"}去。我当时感觉十分惊讶。在无可奈何的情况下，同意他放半年假，半年后再加入。就这样网站就剩下了我与[C.T.]{lang="EN-US"}在孤军奋斗，[Summer]{lang="EN-US"}在业余的时候也经常来发一些贴子，有关[“SQL”]{lang="EN-US"}的讲座也深受大家的欢迎。但一个网站在仅剩下两个人维持的情况下，大家可想而知，作为站长，我的心情是如何的。[]{lang="EN-US"}

[ ]{lang="EN-US"}再加上后来听同事们讲一些中关村小公司的失败，我知道了做一个网站，仅靠朋友义气是不会成功的。[Summer]{lang="EN-US"}在合作的时候，一直在说是尽朋友义务。没有当初的朋友关系，不会走到一起，但如果仅仅为了朋友义气，而没有责任感，朋友做不了，我就[“byebye",]{lang="EN-US"}我想什么也做不了的。不知道大家是否同意我的想法。景儿曾经给我说过，十分同意我在[“]{lang="EN-US"}网站策划书[”]{lang="EN-US"}里关于[“]{lang="EN-US"}义气[”]{lang="EN-US"}与[“]{lang="EN-US"}责任[”]{lang="EN-US"}的论述。关于这点，我也不想多说，大家环顾商海，有多少公司是由几个朋友当初击掌成立，而后又分道扬镖的。[]{lang="EN-US"}

[ ]{lang="EN-US"}在一个多月以前，我曾经相当激动得而且不止一次得告诉他[Summer,]{lang="EN-US"}让他以后不要再说[“]{lang="EN-US"}为朋友尽义气[”]{lang="EN-US"}，这个网站不是我的，虽然我现在是站长，也是这个网站的投资者，但是我不想把这个网站看作自己的，网站的成功与否不只是我个人的成败荣辱，也是我们大家的，当然，现在我还没有能力去理顺我们之间的关系。[]{lang="EN-US"}

[ ]{lang="EN-US"}一个月以前，那天晚上，与[summer,C.T.,]{lang="EN-US"}踏雪，[nd4love,ifonly]{lang="EN-US"}等几个人在午夜剧场碰了一下头，主要讲如何理顺大家的合作关系，大家出了好多好点子，最后责成我写一个策划书。我拜托[FJFJ]{lang="EN-US"}等人帮我在网上找一些这些资料，也看了很多现代管理制度的书，可是我感觉这些知识已经脱离了实际，不太适应现在的网站经营模式，尤其是我们网站，全是在职人员，至少在半年内，大家不会聚到一起来共创一番事业的。我总在想，如何才能够自动量化每个人做的贡献，而不用由某个人来指定，说你这个程序为这个网站做了多大的贡献。并且能够把这种贡献转化为大家的股份，根据股份来选择站长之类的管理者。[]{lang="EN-US"}

[ ]{lang="EN-US"}如果我说我好累，我不知道大家怎么想，我真的想诉一下苦，上一次诉苦后，幸蒙[WebRain]{lang="EN-US"}师兄及各位盟友对我的支持，让我坚持到现在。[]{lang="EN-US"}

[ ]{lang="EN-US"}我其实还是很喜欢编程的，不只是因为有意思，而且成功了，或者失败了，根据网友们的评价就可以知道。我不太喜欢去管理一个网站，因为牵扯到许多人际关系。而这是开心的弱项，开心在网上是一个十分贫嘴而且调皮的男孩，我在现实中其实还是有一些腼腆，有时候还会脾气火暴，可能是痞子蔡所说的双重性格吧。我知道我的性格有时候可能会得罪很多朋友，所以我不愿意坐在协调人的位置上。[]{lang="EN-US"}

[ ]{lang="EN-US"}这个网站是我的希望，我知道这个，所以我也愿意为它卖命得工作，让他取得大家的欢心。老实说，做这个网站后，的确有不少公司的人拉拢过我去加盟，我也曾经心动过，但是我不想放下这个网站。我半年到一年内将会全职来搞这个网站的。这是我的理想，也是我的生活。[]{lang="EN-US"}

[ ]{lang="EN-US"}不知道自己说了些什么，权当牢骚吧，大家别往心里去。[]{lang="EN-US"}
